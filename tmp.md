# Псевдокод

Введем понятие абстрактного шаблона. Это вариант языка со сбалансированными скобочными словами.
Помимо скобок, в языке находятся s, e, t символы, а также r-символы. $r \in R$ --- простое регулярное выражение,
альтернатива по множеству символов из заданного алфавита.

Будем рассматривать шаблоны вида
... e ... e ...
... e ...
e ...
... e
...
Где точки === e без скобок не встречается

Введем начальный алфавит {s, b}. s соотвествует символу (s-переменная),
b соотвествует произвольному выражению в скобках "(e_)".

```
Функция toRegex:
Плоский абстр. шаблон -> регулярка
---

// В алфавите всегда есть символ s (для s-переменной)

// шаблон это список элементов
elements = элементы шаблона
если elements пуст
    возвратить пустую строку
если elements[0] --- r-символ
    возвратить r-символ + toRegex(elements[1:])
если elements[0] --- s-символ
    возвратить s + toRegex(elements[1:])
если elements[0] --- e-символ
    возвратить ".*" + toRegex(elements[1:])
если elements[0] --- t-символ
    возвратить "." + toRegex(elements[1:])
```

```
Функция toFlat
Рефал-функция особого вида -> 
---
Введем начальный алфавит Alphabet = {s, b}. s соотвествует символу (s-переменная),
b соотвествует произвольному выражению в скобках "(e_)".


Пока все шаблоны не стали плоскими
    Найдем выражения с наибольшей скобочной вложенностью во всех шаблонах
    Пронумеруем такие выражения номерами 1, 2, 3, ..., n
    expr1, expr2, ..., exprn = (выражения с наибольшей скобочной вложенностью)
    reg1, reg2, ..., regn = map(toRegex, (expr1, expr2, ..., exprn))
    Для всех reg_i построим минимальный ДКА Q_i, распознающий reg_i в алфавите Alphabet
    Объединим все ДКА Q_i в один ДКА Q с аннтоациями A на T \subset Q
    аннтоация a \in 2^{1, 2, ..., n} в состоянии t озночает, что это состояние принимает языки reg_i для всех i \in a
    Рассмотрим множество различных аннтоаций на T и упорядочим его.
    Обозначим его A'
    k = |A'|
    Новый алфавит Alphabet' = {s} \cup {A'[i] | i \in 1, 2, ..., k}
    Поставим в соотв. expri r-символ вида (A'[j1] | ... | A'[jm]), при этом A'[jl] содержится в альтернативе только тогда, когда i \in a[jl]

    Заменим в шаблонах выражения с наибольшей скобочной вложенностью на соответствующие им r-символы
    Получим абстр. шаблоны с меньшей на 1 скобочной вложенностью
    Alphabet = Alphabet'
В плоских шаблонах заменим пошаблоны без e-символов на регулярные выражения с помощью toRegex

Возвратить полученные переписанные абстр. шаблоны и алфавит Alphabet'
```

Alt+ e Alt+ e Alt+

Где Alt - разные альтернативы вида (a1 | ... | an), где a1, ..., an - символы алфавита, получившегося на последнем итерации toFlat
Любому Alt соответствует множество букв, которые могут быть "реализацией" альтернативы. Обозначим его Set(Alt).

Посл-ть сконкатенированных Alts = Alt1...Altn задает мн-во S = Set(Alt1) \times ... \times Set(Altn). Эл-ту мн-ва тривиально соовтесвует слово w.
Мн-во слов обозн. W(Alts)

Altsi ниже - посл-ти сконкатенированных альтернатив
